# Leap Year

## Instructions

Write a program to find out if given input is a leap year or not

## Example Input

```
input_number = 2000
```

## Example Output

```
2000 is a leap year
```

## Example Input

```
input_number = 1999
```

## Example Output

```
1999 is not a leap year
```
